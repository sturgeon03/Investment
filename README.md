# Investment

This repository is organized by market so the research process stays isolated.

- `us_stocks/`: US equities baseline pipeline, with room for LLM, deep learning, and RL extensions.
- `kr_stocks/`: reserved for a later Korean equities pipeline.
- `project_memory/`: persistent project direction, current state, and Codex operating context.

Start with the US workflow first, prove the research loop, then clone the structure for Korea with local data sources and market-specific execution rules.
